Personal portfolio and CV site. Built with React & TypeScript

---

### Features

- Light and dark mode with system preference detection
- Interactive experience timeline
- Scroll-triggered animations
- Fully responsive magazine-style layout
- Downloadable CV

### Stack

**React 19** &middot; **TypeScript** &middot; **Vite** &middot; **Framer Motion** &middot; **CSS Modules**

### Getting started

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
npm run preview
```
